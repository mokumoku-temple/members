# はじめに
フロントエンドかけこみ寺の自己紹介用リポジトリです。

## もくもく寺とは
フロントエンドでもくもくしたい人たち(エンジニア・デザイナー・ディレクター職種限らず)が
隔週水曜に集まってもくもくする会です。
リモートでの参加も可能です。
